LOCK Example

/cgi-bin/wdwebcgi.exe?LOCK+wd_SID={{session}}+wd_List_ID={{listid}}+wd_List_RecNum={{recnum}}

LOCK Parameters

wd_SID

The Worldox session ID

Optional:

wd_List_ID

The active list

wd_List_RecNum

                The record to lock
